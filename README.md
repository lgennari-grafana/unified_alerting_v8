## Motivation <!-- [![](./docs/img/pin.svg)](#motivation) -->

Custoner Support needs to be constantly updated and trained about new feature relased with each new Grafana version.
This enablement has the the following purposes:
* _Basic knowledge of Unified Alerting_, Unified Alerting is the new version of Alerting system released with Grafana V8.
* _News on Alerting_, Differences between the legacy alerting system and the new unified alerting release.
* _Alert Architecture_, How alert system works and what are the main components that build it
* _Type of Rules_, How many types of ruleas are available to create an alert and how do they work, differ and features.
* _Troubleshooting & Debug_, A chapter for troubleshooting and debug has been added to the enablement. This is based on basic troublshooting rules to follow and best practices.
